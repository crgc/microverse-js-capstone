## Getting Started

### Prerequisites

- [Webpack](https://webpack.js.org/)

### Setup

* Clone this repository
```
git clone https://github.com/crgc/microverse-js-capstone.git
```
* Change into folder
```
cd microverse-js-capstone
```
* Install dependencies
```
npm install
```

### Run
* Build
```
npm run build
```
* Start server
```
npm start
```
* Open your browser at http://localhost:8080/

### Tests
* Run Jest test suite
```
npm test
```
